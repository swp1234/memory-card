# Memory Card Flip - 메모리 카드 플립 게임

카드를 뒤집어서 짝을 맞추는 중독성 높은 기억 게임입니다. 콤보 시스템과 다양한 난이도, 테마로 더욱 즐거운 게임 경험을 제공합니다.

## 게임 기능

### 핵심 게임플레이
- **카드 매칭**: 카드를 뒤집어서 같은 그림 2개를 맞추세요
- **스테이지 시스템**: 난이도가 점점 상승하는 4개 스테이지
- **콤보 시스템**: 연속으로 매칭할 때 콤보 계수 증가 → 높은 점수 획득
- **점수 체계**: 기본점수(100) + 콤보보너스 + 시간보너스

### 게임 난이도
1. **쉬움 (4x3)** - 12개 카드 (6개 쌍)
2. **보통 (4x4)** - 16개 카드 (8개 쌍)
3. **어려움 (5x4)** - 20개 카드 (10개 쌍)

### 테마 선택
1. **이모지** 😊😎🤩 - 감정 표정 이모지
2. **동물** 🐶🐱🐭 - 귀여운 동물 캐릭터
3. **과일** 🍎🍊🍋 - 신선한 과일들
4. **국기** 🇰🇷🇺🇸🇯🇵 - 세계 여러 나라의 국기

### 상호작용
- **일시 정지**: 게임을 언제든지 멈출 수 있음
- **결과 공유**: SNS에 점수와 성과를 공유
- **최고 기록**: 최고 점수를 자동으로 저장 (localStorage)
- **다국어 지원**: 12개 언어 완벽 지원

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - Glassmorphism, 3D 카드 플립, 반응형 디자인
- **Vanilla JavaScript** - 게임 로직, i18n, 음향효과
- **Web Audio API** - 게임 사운드 효과
- **Service Worker** - PWA (오프라인 지원)

## 디자인 원칙

### 2026 UI/UX 트렌드 적용
- **Glassmorphism** - 반투명 배경, 블러 효과로 모던한 느낌
- **Dark Mode First** - 어두운 배경(#0f0f23)을 기본으로 설정
- **Microinteractions** - 부드러운 카드 플립, 파티클 이펙트
- **Minimalist Design** - 깔끔한 레이아웃, 충분한 여백
- **접근성** - 터치 타겟 44px+, 명확한 색상 대비

### 색상 체계
- **주색상**: #e67e22 (오렌지) - 따뜻하고 친근한 느낌
- **배경**: #0f0f23 (진한 파란-검정) - 눈 피로 최소화
- **강조색**: #f39c12 (황금색) - 콤보, 보너스 표시

### 애니메이션
- **카드 플립**: 3D 회전 애니메이션 (600ms)
- **파티클**: 매칭 성공 시 별 파티클 생성
- **점수 팝업**: 획득 점수 시각화
- **로딩**: 부드러운 페이드인/아웃

## 게임 로직

### 점수 계산
```
기본점수 = 100
콤보보너스 = 콤보 수 × 10
시간보너스 = (300 - 경과시간) × 10 (최소 0)
총점 = 매칭점수 + 시간보너스 + 콤보보너스
```

### 스테이지별 난이도 증가
- Stage 1: 4x3 (12개, 쉬움)
- Stage 2: 4x4 (16개, 보통)
- Stage 3: 5x4 (20개, 어려움)
- 그 이상: 난이도 유지

### 게임 오버 조건
- 모든 스테이지 완료 후 quit 버튼 클릭
- 또는 3회 연속 실패 (추가 구현 가능)
